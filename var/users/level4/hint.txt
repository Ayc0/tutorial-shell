Try to look what is in your SQLite database and to parse the CSV file

* https://docs.python.org/3/library/sqlite3.html
* https://docs.python.org/3/library/csv.html

We've created a small command "run" that allows you to run simple sql queries.
For instance:
* 'run "show tables"' to list every tables
* 'run "pragma table_info(<table_name>)"' to see the definition of a table
