* Find the root's password
* Find the command "su"

Once you found the password and the command, you'll have access to the user "root" which is the super administrator of this machine.

As usual, you can type "su root" or in this case just "su".

The goal of this level is to understand the PATH and Linux filesystem (https://linuxconfig.org/linux-path-environment-variable and https://serverfault.com/questions/384342/what-are-the-best-practices-of-the-usr-var-and-etc-folders)
